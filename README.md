# Nexus AI – Intelligent Customer Support System

An emotion-aware AI customer support chatbot built using **Groq API** and **Streamlit**.  
The system detects user sentiment and provides intelligent responses.

## Features
- Emotion-aware chatbot
- Real-time conversation handling
- Escalation detection for frustrated users
- Conversation history storage
- Interactive UI using Streamlit

## Technologies Used
- Python
- Streamlit
- Groq API
- Sentiment Analysis
- JSON Database

## Project Structure
```
app.py
database.py
sentiment_analyzer.py
escalation.py
animations.py
requirements.txt
```
